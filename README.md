# Machine Learning Based Spectrum Sensing

## Overview
This project uses machine learning techniques to classify spectrum channels as free or occupied in cognitive radio networks.

## Models Used
- Logistic Regression
- Decision Tree

## Features
- Signal Strength
- SNR (Signal-to-Noise Ratio)
- Energy
- Primary User Presence

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- Precision-Recall Curve

## Results
Achieved high classification accuracy (~94–96%) with effective performance across models.

## Tools & Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure
- Dataset: Contains 10,000 samples of spectrum data
- Code: Implementation of ML models
- Output: Performance graphs and evaluation metrics
